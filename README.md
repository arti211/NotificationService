# Notification Service

## Purpose
The Notification Service is designed to handle notifications for various applications. It provides a robust and scalable way to send notifications through different channels.

## Tech Stack
- **Java**: The core programming language used.
- **Spring Boot**: For building the RESTful web services easily.
- **Maven**: To manage the project dependencies and build process.

## How to Run
1. Clone the repository:
   ```bash
   git clone https://github.com/arti211/NotificationService.git
   cd NotificationService
   ```
2. Build the project using Maven:
   ```bash
   mvn clean install
   ```
3. Run the application:
   ```bash
   mvn spring-boot:run
   ```
4. Access the service at `http://localhost:8080/notifications`.
